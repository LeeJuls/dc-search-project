# 프로젝트 히스토리: 크롤러 개발 및 테스트 (Phase 2)

## 📅 일시: 2026-03-16
## 📜 버전: v1.1

### 1. 개발 내용
- **모듈명:** `dc_crawler.py` (Backend)
- **주요 기능:**
    - 디시인사이드 목록 페이지(`us-post`) 데이터 추출.
    - `get_standard_date` 함수를 통한 날짜 표준화 (YYYY-MM-DD HH:mm:ss).
    - 차단 방지를 위한 `random.uniform` 지연 시간 적용.
    - 초보 사용자 이해를 돕기 위한 코드 주석 처리.

### 2. 테스트 결과 (QA)
- **테스트 대상:** 쿠키런 오븐스매시 마이너 갤러리 1페이지.
- **성공 항목:**
    - HTTP 200 정상 응답 수신.
    - 총 45개의 게시글 데이터 추출 완료.
    - 날짜 변환 성공: `2026-03-16 14:52:35` 형식으로 정확히 파싱됨.
- **발견된 이슈 및 조치:**
    - 초기 테스트 시 날짜가 중복 결합되는 현상 발견 -> `get_standard_date` 로직 수정으로 해결 (세이브 포인트).

### 3. 향후 과제
- 수집된 데이터를 영구 저장하기 위한 SQLite DB 설계 (Phase 3).
- 1주일치 데이터를 안전하게 소급해서 가져오는 루프 로직 최적화.
